Product FAQs configure

After you install this app, That you finished setup and will be redirect to store's products table.


All products's questions

You can create new question



You can publish/unpublish or delete questions for one product.


Click to product's title, page will redirect to detail questions page of this product.

You can:

  • Reply/Edit questions
  • Publish/unpublish, lock/unlock or delete with selected questions

Menu:

  • Home: Go to app admin page
  • Edit product: Go to Shopify's product page to edit product
  • View on store: Vie product page on your store to see reviews's list

Products FAQs App UnInstallation

If you want to uninstall this app, following these steps:

Step 1: Go to Edit HMTL page of your theme: Online Store -> Themes -> Edit code


Step 2: Go to Edit theme template: Sections -> product-template.liquid, remove code you added such as <div class="ot-product-faqs"></div>
